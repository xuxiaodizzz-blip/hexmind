# Security Policy

## Supported Scope

This public repository covers the open-core HexMind codebase and sample assets.

Please report:

- credential leaks
- authentication flaws
- unsafe default behavior
- dependency vulnerabilities that materially affect the public repo

## Reporting

For sensitive issues, please avoid opening a public issue with exploit details.

Use GitHub's private vulnerability reporting flow if it is enabled for the repository. If private reporting is not available, contact the maintainer through the repository owner's GitHub profile first, and only open a public issue after the risk is understood.

For non-sensitive bugs, please use the standard issue templates.
