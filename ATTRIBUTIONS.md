# Attributions

HexMind is an original product, but parts of the implementation build on widely used open-source projects that are hosted on GitHub.

The current codebase uses these projects directly for core product capabilities:

- `FastAPI`: API framework and request handling for the backend service
- `SQLAlchemy`: async database models and repository layer
- `LiteLLM`: model-provider abstraction and unified LLM calling
- `Instructor`: structured output workflows around model responses
- `Click` and `Rich`: CLI command surface and terminal presentation
- `React` and `Vite`: web application runtime and frontend build tooling
- `Tailwind CSS`: utility-first styling in the web UI
- `Framer Motion` and `motion`: page transitions and interface animation
- `Recharts`: charts used in analytics views
- `Lucide React`: icon set used in the frontend

This repository also includes compatibility points and product ideas informed by the broader open-source ecosystem, but the HexMind decision flow, asset model, and product packaging are maintained in this project.

Please review each dependency's own repository and license for its specific terms.
